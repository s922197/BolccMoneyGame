# 🏥 社畜生存戰：理財 Roguelike

## 🎮 遊戲簡介

「社畜生存戰：理財 Roguelike」是一個結合了**資產管理**、**健康維持**與**隨機事件**的雙人回合制網頁遊戲。玩家將扮演現代社畜，在有限的 20 個回合內，通過選擇工作、投資和兼職等行動，努力平衡現金流、累積總資產，同時應對來自過勞和突發帳單的健康與財務挑戰。

**最終目標：** 在 20 輪遊戲結束時，總資產（現金、存款、投資）最高且健康狀況良好的玩家獲勝。

---

## ✨ 遊戲特色

### 1. 創新的回合制決策

* **多重選擇：** 每輪從 5 張隨機生成的行動卡中，選擇 **1 至 2 張**卡片執行。
* **時間壓力：** 隨著遊戲進展，玩家的決策時間將逐漸縮短，考驗快速反應與策略規劃能力。

### 2. 高風險的金融陷阱

* **整合詐騙卡：** 高風險詐騙卡片（如虛擬幣投資、高報酬保單）已完全整合到常規卡池中，隨時可能出現。
* **強制損失機制：** 選擇詐騙卡片將會導致**直接的巨額損失**，損失額度可能導致存款變為負債。

### 3. 殘酷的健康與過勞系統

* **健康值 (HP)：** 選擇高工時工作或兼職將導致健康值下降，工作時長超過 8 小時即視為過勞。
* **疾病懲罰：** 健康值過低時，將有高機率累積醫療費用（病債）。
* **強制治療：** 當健康值低於危險線時，玩家必須在下一輪**強制選擇治療卡**，否則無法進行其他行動。

### 4. 豐富的資產管理線

* **多元收入：** 穩定工作、兼職、股利、隨機獎金等。
* **理財工具：** 活期存款、固定期限存款（有鎖定期）、短期高風險投資、藍籌股投資。
* **隨機事件：** 每輪結算時，有隨機機率觸發額外的小型收入或帳單（例如：繳稅、紅包、罰單）。

---

## ⚙️ 遊戲規則與機制

### 🕹️ 行動階段

* **輪次：** 總計 20 輪，每輪包含 A 組行動和 B 組行動。
* **選擇數量：** 第 1 輪只能選擇 1 張工作卡；第 2 輪起，每輪可選擇 2 張卡片。
* **存款特殊處理：** 選擇「彈性存款」會立即彈窗要求輸入存款比例，不佔用卡片選擇次數。
* **執行順序：** A 組行動 -> B 組行動 -> 結算。

### 💰 結算機制

* **自動扣除：** 每輪自動扣除 **生活費 ($30)**，並根據當輪工作總收入額外扣除 **10%** 作為奉獻/稅金。
* **扣款優先順序：**
    1.  **卡片成本/詐騙損失：** 優先扣除現金，現金不足則扣存款，允許存款變為負債（僅限強制損失）。
    2.  **結算時支出 (生活費/奉獻)：** 優先扣除現金，現金不足則扣存款，允許存款變為負債。
* **投資結算：** 短期投資和定存會在指定輪次後自動結算盈虧或領回本金。

## 🚀 未來潛在擴展 (Future Scope)

1.  **市場波動性：** 引入大盤指數，讓投資（如藍籌股）的價值隨著市場波動，增加不確定性。
2.  **債務與信用：** 當銀行存款變為負數時，引入「債務利息」機制，讓玩家需償還利息。
3.  **房地產系統：** 增加長期低流動性的房地產投資卡，提供租金收入和潛在的資產增值。
4.  **技能與天賦樹：** 引入經驗值，讓玩家可以投資於技能，提高工作收入或投資成功率。
